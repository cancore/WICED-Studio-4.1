Enumeration Type Documentation

enum wiced_debug_uart_types_t

Debug UARTs.

Used when calling wiced_set_debug_uart.

Enumerator
WICED_ROUTE_DEBUG_TO_WICED_UART 

Set to WICED UART to send debug strings over the Wiced debug interface.

WICED_ROUTE_DEBUG_TO_HCI_UART 

By default, HCI UART will be used if the application doesn't set any other using wiced_set_debug_uart()

WICED_ROUTE_DEBUG_TO_DBG_UART 

Use this to direct the debug traces over the debug uart for the 20 MHz board.

WICED_ROUTE_DEBUG_TO_PUART 

Use this to direct the debug traces over the peripheral uart for the 24 MHz board.
