void a4wp_hal_i2c_mux_configure ( void  )

Function a4wp_hal_i2c_mux_configure.

Configure the Serial flash lines to I2C.

Use a4wp_hal_i2c_mux_configure to configure the lines for I2C operation prior to starting any HAL initializations.

The I2C and Serial flash lines are multiplexed in 20703. Serial flash is the default mode when detected with boot up. So when the patch is loaded from Serial flash, it has to be re-configured to use with I2C lines (to communicate with BRCM PMU).

Returns
void

Function a4wp_hal_init.

Allows the app to configure and initialize the PMU interfaces.

Use a4wp_hal_init to setup the WPT Interrupt line with the GPIO and to enable the Iout/Irect algorithm for 59350 PMU.

Parameters
[in]wpt_int_gpio: GPIO configured for WPT Interrupt.
[in]enable_i_out_wa: Enables the Iout/Irect algorithm for 59350 PMU. This should be WICED_FALSE for other PMUs
Returns
PMU device Id.
