uint8_t* wiced_raw_uart_get_read_data_position ( uint32_t *  p_data_len)

Gets the received data from UART to the application.

This function returns the UART data pointer and the available data length. Application should invoke wiced_raw_uart_update_read_data_position() after reading the data with the processed data length for the UART driver to update the data pointer.

Parameters
[out]p_data_len:Length of the available UART data
Returns
the pointer to the UART data, NULL if no data available

wiced_bool_t wiced_raw_uart_update_read_data_position ( uint32_t  read_len)

Application uses this function to update the read data length and this function returns TRUE if more data is available with the UART driver, false otherwise.

Application should invoke this after reading the data for the UART driver to update the data pointer.

Parameters
[in]read_len:Length of the data read by the application
Returns
TRUE if more data is available with the UART driver, FALSE otherwise
