Detailed Description

Message used to forward an MPAF packet from any transport to the HCI thread.

Such a message should have its msg.code field set to TODO-TBD, should be allocated using TODO-TBD and released using TODO-TBD. Data begins immediately after this structure, which has a size of 12 bytes. That way, the data will be aligned to a 32-bit word boundary to accommodate DMA for transfer of the data payload.
