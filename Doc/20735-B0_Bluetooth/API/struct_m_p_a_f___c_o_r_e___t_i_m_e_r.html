Detailed Description

Message used to pass messages from applications to applications the applications can use this message header to prepend with its message.

The length field is used by the application to identify and parse the message.

Field Documentation

MPAF_TIMER_TYPE type

Type to differentiate the timer characteristics If periodic MPAF reloads the timer in every overflow.

If Mpaf context then the MPAF changes the context of the timer execution.
