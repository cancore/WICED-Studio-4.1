A structure used for an entry in FOUNDATION_CONFIG_ITEM_ID_32BIT_REGISTER_POKES.

Field Documentation

UINT32 mask

The mask of value bits that are to be written.

Where mask bits are 1, the corresponding value bit will be applied. Where mask bits are 0, the existing bit value in the target hardware register will be preserve.
