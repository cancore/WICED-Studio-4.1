zb_zdo_ret_type_t(* ctrl_req)(zb_zdo_ctrl_req_t *req)

Function for APP layer as the ZDO CTRL interface.


It creates a a new ZDO_CTRL request (CTRL_REQ)

Parameters
[in,out]reqpointer to ZDO CTRL request parameter
Return values
ZDO_RET_SUCCESSsuccessful
ZDO_RET_SUCCESS_RESPsuccessful with response
othersfailed
Note
When retval is ZDO_RET_SUCCESS_RESP, ZDO will call the response function registered in ctrl_resp_reg() to send ZDO_CTRL response
