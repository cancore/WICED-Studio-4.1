enum zb_zdo_node_state

ZigBee network node state declaration.

Enumerator
IN_NETWORK 

The node is connected to Zigbee network.

OUT_OF_NETWORK 

The node is disconnected from Zigbee network.

OUT_OF_NETWORK_REJOIN 

The node is temporary disconnected and rejoining.

OUT_OF_NETWORK_LEAVE 

Out of Network due to NWK leave.
