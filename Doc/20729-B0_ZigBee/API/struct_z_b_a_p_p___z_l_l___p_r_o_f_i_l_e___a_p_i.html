uint8_t(* selTargetNode)(uint16_t panid)

API: After a scan request command, AO may receive many responses.


AO can use this function to select which one as target to unicast following commands.

Parameters
[in]panid- AO chooses the target which PAN ID is equal to the input of API.
If AO set panid as 0, the first target in the responder list will be selected.
Returns
The status of matching target node
Return values
0fail
1success
