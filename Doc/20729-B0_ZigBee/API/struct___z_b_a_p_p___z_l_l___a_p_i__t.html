nwkjoin_rt_req: This is a function pointer
AO can use this function to unicast a network join router request to a specific target.

Parameters
[in]assignNwkAddrAO uses it to assign the ¡§Network address field¡¨ in payload as a new nwkAddr to target.
If this parameter is NULL, the default nwkAddr will be assigned to the field.
[in]panIdAO uses it to assign the ¡§PAN identifier field¡¨ in payload as a new PAN ID to the new network.
If this parameter is NULL, the current PAN ID will be assigned to the field.
[in]extPanIdAO uses it to assign the ¡§Extended PAN identifier field¡¨ in payload as a new extended PAN ID to the new network.
If this parameter is NULL, the current extended PAN ID will be assigned to the field.
