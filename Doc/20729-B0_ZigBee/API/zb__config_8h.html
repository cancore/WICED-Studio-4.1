SPEC defined clst attr - Temp Meas cluster.

SPEC defined clst attr - scene cluster.

SPEC defined clst attr - OTA cluster.

SPEC defined clst attr - On / Off cluster.

SPEC defined clst attr - level cluster.

SPEC defined clst attr - identify cluster.

SPEC defined clst attr - Group cluster.

SPEC defined clst attr - Color Control Cluster.

brief SPEC defined clst attr plus supportability for single attribute - Basic cluster

The config variable is g_zbstack_config. Config item code for ZigBee basic cluster srv init parameter. The config variable is g_zcl_srvAttrInit_basicClst Config item code for ZigBee color ctrl cluster srv init parameter. Master Key used for frame security.

If all zero, no master key will be used for security. If this is set, the master key will be used for ZigBee security process.

Maximum binding permitted.

The value used for max. bind allowed in a ZigBee device.

The specific 64-bit extended PAN ID to form network or to connect to.

If set to all zero, PAN coordinator will set its IEEE address to the extended PAN ID. It's suggested to use all zero for ZC to prevent conflict. If set to all zero, ZigBee devices (not including ZC) will try to connect to any available networks.

Set the device to work as ZigBee Coordinator.

This value is valid only when deviceRole is set to FFD (0x02).

The available channel for ZigBee network.

True if the device is allowed to perform insecure join, else false.

The threshold value for parent link retry while fail.
