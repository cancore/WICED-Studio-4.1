enum _ZBAPP_DST_NODE_ACT

Actions for handling destination node.

Enumerator
DEV_NODE_ACT_REMOVE 

Remove a destination node.

DEV_NODE_ACT_REMOVE_ALL 

Remove all destination nodes.

DST_NODE_ACT_SET_MAX_RX_BUF_SIZE 

Set maximum receiving buffer size of a destination node.

ZBAPP_DATA_STATUS zbapp_txDataDel ( uint8_t  aoId,
ZBAPS_APSDE_PRIMITIVE_t *  pRxDataPriv,
uint8_t  dataType 
)

Delete the "TX data node" by providing the received DATA CNF or RSP.


If DATA CNF is provided, it can find DATA REQ of any type.
If DATA RSP is provided, it depends on the provided cb of ZBAPP_FIND_REQ_FROM_RSP.

Parameters
[in]aoIdThe specified AO ID.
[in]pRxDataPrivThe APSDE CNF or RSP to find the "TX data node".
[in]dataTypeIf DATA RSP is provided, one must give dataType and provide cb of ZBAPP_FIND_REQ_FROM_RSP
If DATA CNF is provided, then this value doesn't matter.
Returns
Handling result.
Return values
ZBAPP_DATA_INVALID_PARAMETERThere are invalid parameters.
ZBAPP_DATA_SUCCESS_NO_ACTIONThere are no "TX data node" found.
ZBAPP_DATA_SUCCESS_GENERALThere are "TX data node" found and deleted.
