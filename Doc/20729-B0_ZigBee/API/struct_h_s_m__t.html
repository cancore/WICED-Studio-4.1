Detailed Description

HSM_t is the basical structure (base class in C++).

Hierarchy State Machine The application would derived directly from this structure for different HSM design.

Note
HSM_t used the base class framework in C. It needs to be initialized for function table dynamically in hsm_ctor() function just like constructor function used in O-O language
